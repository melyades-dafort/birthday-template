# 🚀 Quick Start Guide - Selling Birthday Templates

## Your First Sale in 5 Steps!

### Step 1: Get Customer Info
Ask customer for:
- Birthday person's name
- 12 photos
- Birthday message
- Payment (₱300-1500)

### Step 2: Customize
```bash
# Start the development server
npm run dev
```

Go to `http://localhost:8080/admin`:
1. Enter name
2. Upload 12 photos
3. Add titles and messages
4. Write final birthday message
5. Click "Save Changes"

### Step 3: Build Single File
```bash
# This creates one HTML file with everything inside
npm run build:single
```

You'll get a file like: `birthday-john-doe.html`

### Step 4: Send to Customer
- Send the HTML file via Facebook Messenger
- Tell them: "Download and double-click to open!"

### Step 5: Get Paid & Reviews!
- Confirm payment received
- Ask for review/testimonial
- Request referrals

## ❓ Common Questions

**"How big is the file?"**
- Usually 5-15 MB depending on photos

**"Will it work on phones?"**
- Yes! Works on any device with a browser

**"Can they edit it?"**
- No, it's view-only (which is perfect!)

**"How long does it take to make?"**
- 15-30 minutes per order once you get the hang of it

**"Do I need hosting/website?"**
- No! The file works offline, no website needed
