## 🚀 Quick Start (3 Easy Steps!)

### STEP 1: Open the File

1. **Double-click** the `birthday-template.html` file
2. It will open in your **web browser**
3. You'll see a welcome screen

### STEP 2: Go to Admin Panel

**Change the URL to add `/admin` at the end:**

**Before:** 
```
file:///C:/Users/YourName/Downloads/birthday-template.html
```

**After:**
```
file:///C:/Users/YourName/Downloads/birthday-template.html#/admin
```

**OR if you have the online version:**
```
https://yoursite.com/admin
```

### STEP 3: Customize Everything!

You'll see 3 tabs:

#### 📌 **General Tab**
- Enter the birthday person's **name**
- Upload **custom music** (MP3 file) or keep default

#### 📌 **Photos & Captions Tab**
- Upload **12 photos** (memories with the birthday person)
- Add a **title** for each photo (short, 2-5 words)
- Write a **message** for each photo (1-3 sentences)

#### 📌 **Messages Tab**
- Write your final **birthday greeting message**
- This appears at the end of the experience

**Don't forget to click "SAVE CHANGES"!** ✅

## 💡 Tips for Best Results

### Photos:
- ✅ Use **high quality** images (but under 2MB each)
- ✅ Mix different moments (candid, posed, events)
- ✅ Choose photos with **good lighting**
- ✅ Keep aspect ratio consistent (portrait works best)

### Titles:
- ✅ Keep them **short** (2-5 words)
- ✅ Make them **meaningful**
- ✅ Examples:
  - "Our First Adventure"
  - "Best Friends Forever"
  - "Summer Memories"
  - "Unforgettable Moment"

### Messages:
- ✅ Keep them **personal** and **heartfelt**
- ✅ 1-3 sentences per photo
- ✅ Tell a **story** or share a **memory**
- ✅ Final message: 2-4 sentences of birthday wishes

### Music:
- ✅ Choose a song that's **meaningful** to you both
- ✅ Use MP3 format
- ✅ Keep file size under 10MB
- ✅ Make sure you have **rights** to use it
- ✅ Default song is already included!

## ❓ Troubleshooting

### Photos not showing?
- ✅ Check file size (should be under 5MB)
- ✅ Use JPG, PNG, or WebP format
- ✅ Try uploading one at a time
- ✅ Refresh the page after saving

### Music not playing?
- ✅ Ensure file is MP3 format
- ✅ Check if browser allows autoplay
- ✅ Click anywhere on page to start audio
- ✅ Check volume is on

### Changes not saving?
- ✅ Make sure you clicked "Save Changes"
- ✅ Try refreshing the page
- ✅ Check if browser allows localStorage
- ✅ Try a different browser

### File won't open?
- ✅ Make sure you have a web browser (Chrome, Firefox, Safari, Edge)
- ✅ Try right-click → "Open with" → Choose browser
- ✅ Move file out of Downloads if it's blocked

## 🎓 Advanced Tips

### Want to use it multiple times?
- Create a copy of the HTML file for each person
- Rename it (e.g., `birthday-john.html`, `birthday-mary.html`)
- Customize each separately
- Each file stores its own data!

### Want to reset?
- Open browser **DevTools** (F12 key)
- Go to **"Application"** or **"Storage"**
- Find **"Local Storage"**
- Delete the `birthdayData` entry
- Refresh page - it's reset!

### Want to make changes after sharing?
- Just edit in admin again
- Save changes
- Re-send the updated file
- Previous versions won't update automatically
